Reading Passage

Operating systems manage hardware resources and provide services for computer programs.
Efficient OS design ensures better performance, memory management, and process scheduling.

Questions

1. What is the main role of an operating system?
	The main role of an operating system is to manage hardware resources and provide services for computer programs.

2. Why is memory management important?
	Memory management ensures efficient use of memory and improves system performance.

3. What is process scheduling?
	Process scheduling is the method that decides which process runs at a given time.

BLOCK 3 – COMMUNICATION PRACTICE

Speak for 3 minutes on:

Explain how the Internet works

Include:
- What happens when you type a URL
- DNS
- Server
- Response

The Internet is a network that connects computers worldwide.
When you type a URL, your computer sends a request to find that website.
DNS (Domain Name System) changes the website name into an IP address.
The request goes to a server where the website is stored.
The server sends back the data (response).
Your browser shows the webpage on the screen.
