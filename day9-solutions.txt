BLOCK 1 – APTITUDE (Time & Work)

Question 1
A can complete a work in 10 days and B in 15 days.
How many days will they take together?
	
	A’s 1 day work=1/10  
	B’s 1 day work=1/15  
	Together work =1/10+1/15  
                      =(3+2)/30  
                      =5/30  
                      =1/6  
	Total days = 6 days  

Question 2
A alone can do a work in 12 days.
After working for 4 days, B joins and work finishes in 4 more days.
Find B's efficiency.

	A’s 1 day work=1/12  
	Work done by A in 4 days=4/12  
                         =1/3  
	Remaining work=1-1/3  
                      = 2/3  
	(A + B) 1 day work=(2/3)/4  
                     =2/12  
                     =1/6  
	B’s 1 day work=(1/6-1/12)  
                =(2-1)/12  
                =1/12  
	B takes 12 days  

Question 3
If 5 people can complete a work in 20 days,
how many people are needed to complete the work in 10 days?

	Work=People×Days  
	5×20=P×10  
	100=10P  	
	P=10 people  

Question 4
A is twice as efficient as B.
If B takes 12 days to complete work, how many days will A take?

	Efficiency=1/Time  
	A:B=2:1  
	Time ratio=1:2  
	A time=(12×1)/2  
              =6 days  

Question 5
Pipe A fills a tank in 6 hours and Pipe B fills in 8 hours.
How long will both take together?

	A 1 hour work=1/6  
	B 1 hour work=1/8  
	Together work=1/6+1/8  
                =(4+3)/24  
                =7/24  
	Total time=24/7 hours  
                  =3.43 hours

BLOCK 3 – COMMUNICATION PRACTICE

Speak for 3 minutes on:

Explain what happens when you run a program

Include:
- Compilation / Interpretation
- Memory allocation
- Execution
- Output

When we run the program, the first step is compilation or interpretation.
For compiled languages such as C or Java, the program is compiled into machine code before running.
For interpreted languages such as Python, the program is interpreted line by line.
Next is memory allocation.
Memory allocation takes place in the RAM for the variables, functions, and data structures.
This allocation also includes the memory allocation for the stack and the heap.
Then comes the execution part.
The CPU executes the program step by step, performing arithmetic, logical, and control operations.
Then the program produces the output.
The result is displayed on the screen, saved in a file, or returned as a value.
So, the program goes through the phases of compilation, memory allocation, execution, and finally the output.
